# quiz_telegram_bot

quiz_telegram_bot - Программа телеграмм-бота реализующего игру в квиз (вопрос-ответ-реакция)  

 Команды:  

  /start - старт работы бота  

  /quyz - начать квиз  

  /stats - статистика  

  
quiz_data.py - 10 вопросов квиза  

show_db.py - просмотр базы текущих состояний и статистики
